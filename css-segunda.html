  • Estilos de la fuente tipográfica: Propiedades que afectan a la fuente que está siendo aplicada al texto (qué fuente se aplica, como de grande es, si es negrita, itálica, etc).
  • Estilos de disposición del texto: Propiedades que afectan el espaciado y otras característica de la disposición del texto, permitiendo la manipulación de, por ejemplo, el espacio entre líneas y letras, y como el texto se alinea dentro de la caja contenedora.

Estilos de la fuente tipográfica: propiedades

color: Valores de color de CSS


	p {
	  color: red;
	}

font-family:

  • Fuentes seguras para la web:
    • Arial, Courier New, Georgia, Times New Roman, Trebuchet MS y Verdana.
  • Fuentes predeterminadas:
    • serif, sans-serif, monospace, cursive y fantasy.

Listas de fuentes: Ya que no puede garantizar la disponibilidad de las fuentes que desea utilizar, puede proporcionar un lista de fuentes para que el navegador tenga múltiples fuentes entre las que elegir. Esto simplemente involucra al valor de font-family que debera consistir de una lista de varios nombres de fuentes separados por comas.


	p {
	  font-family: Helvetica, Arial, sans-serif;
	}

font-size Tamaño de fuentes:

px, em, rem y %

CSS proporciona cuatro propiedades comunes para alterar el peso visual/énfasis del texto:

  • font-style: Usado para activar y desactivar el texto en cursiva.
    • normal: Establece el texto de la fuente normal (desactiva la cursiva existente).
    • italic: Establece el texto para usar la versión en cursiva de la fuente si está disponible; si no está disponible, simulará cursiva con oblicuo en su lugar.
    • oblique: Establece el texto para usar una versión simulada de cursiva en la fuente, creada inclinando la versión normal.
  • font-weight: Establece como negrita el texto. Este tiene muchos valores disponibles en caso de que tenga muchas variantes de fuente disponibles (tales como -light, -normal, -bold, -extrabold, -black, etc.):
    • normal, bold: Peso de la fuente Normal y negrita
    • lighter, bolder: Establece la negrita del elemento actual para que sea un paso más claro o más pesado que la negrita de su elemento padre.
    • 100–900: Valores numéricos de negrita que proporcionan un control más detallado que las palabras claves anteriores, si es necesario.
  • text-transform: Le permite configurar su fuente para ser transformada. Los valores incluyen:
    • none: Evita cualquier transformación.
    • uppercase: Transofrma TODO EL TEXTO A MAYÚSCULAS.
    • lowercase: Transforma todo el texto en minúsculas.
    • capitalize: Transforma todas las palabras para Tener La Primera Letra En Mayúscula.
    • full-width: Transforma todos los glifos que se escribirán dentro de un cuadrado de ancho fijo, similar a una fuente monoespaciada, lo que permite la alineación de por ejemplo, caracteres latinos junto con glifos del lenguaje asiático (como Chino, Japonés, Koreano.)
  • text-decoration: Activa/desactiva las decoraciones de texto en las fuentes (principalmente usará esto para desactivar el subrayado por defecto en los enlaces al diseñarlos.) Los valores disponibles son:
    • none: Desactiva cualquier decoración de texto que ya esté presente.
    • underline: Subraya el texto.
    • overline: Proporciona al texto una línea superpuesta.
    • line-through: Coloca un tachado sobre el texto.

Debe tener en cuenta que text-decoration puede aceptar múltiples valores a la vez, si desea agregar múltiples decoraciones simultaneamente, por ejemplo text-decoration: underline overline. También tenga en cuenta que text-decoration es una propiedad abreviada de text-decoration-line, text-decoration-style, y text-decoration-color. Puede utilizar combinaciones de estos valores de propiedad para crear efectos interesantes, por ejemplo text-decoration: line-through red wavy.

text-shadow:


	p {
		text-shadow: 4px 4px 5px red;
	}

Puedes aplicar múltiples sombras al mismo texto incluyendo multiples valores de sombra separados por comas.

Estilos de disposición del texto: propiedades.

La propiedad text-align: left, right, center y justify.

La propiedad line-height establece la altura de cada línea de texto — esto puede tomar la mayoría de las unidades y magnitudes, pero tabién puede tomar un valor sin unidades, que actúa como un multiplicador y generalmente se considera la mejor opción — la propiedad font-size es multiplicada para obtener la altura de la línea (line-height).

Las propiedades letter-spacing y word-spacing le permiten establecer el espaciado entre letras y palabras en el texto.

Forma abreviada de Font: Muchas propiedades de fuente también pueden ser establecidas de forma breve a través de la propiedad font. Estas son escritas en el siguiente orden: font-style, font-variant, font-weight, font-stretch, font-size, line-height, and font-family. De entre todas estas propiedades, solo son requeridas font-size y font-family cuando se usa la propiedad font de forma abreviada. Se debe colocar una barra inclinada (slash) entre las propiedades font-size y line-height. Un ejemplo completo se vería así: 


	font: italic normal bold normal 3em/1.5 Helvetica, Arial, sans-serif;

Estilos específicos de las listas:

  • list-style-type: Establece el tipo de viñetas para la lista, por ejemplo, disc, circle, square, viñetas cuadradas o circulares para una lista desordenada; decimal, decimal-leading-zero, lower-roman, upper-roman, lower-greek, lower-latin, upper-latin, armenian, georgian, lower-alpha, upper-alpha, números, letras, o números romanos para una lista ordenada.
  • list-style-position: Establece si las viñetas aparecen dentro de los elementos de la lista, inside, o fuera de ellos antes del inicio de cada elemento, outside, comportamiento por defecto.
  • list-style-image: Le permite usar una imagen personalizada para la viñeta, en lugar de un simple cuadrado o círculo.

La propiedad list-style-image le permite usar una imagen personalizada para su viñeta. Sin embargo, esta propiedad es un poco limitada para controlar la posición, tamaño, etc. de las viñetas. Es más conveniente usar la familia de propiedades background.


    ul li {
	  padding-left: 2rem;
	  background-image: url(star.svg);
	  background-position: 0 0;
	  background-size: 1.6rem 1.6rem;
	  background-repeat: no-repeat;
	}

Las tres propiedades mencionadas pueden ser declaradas en una sola propiedad, list-style. Por ejemplo:


    ul {
	  list-style-type: square;
	  list-style-image: url(example.png);
	  list-style-position: inside;
	}

Se convierte en:


    ul {
	  list-style: square url(example.png) inside;
	}

Lo primero para entender el concepto de el estado de enlace - diferentes estados que los enlaces pueden tener, que pueden ser establecidos usando diferentes pseudo-clases:

  • Link (no visitado): El estado por defecto en que se encuentra un enlace, cuando no está en ningún otro estado. Este puede ser especificamente diseñado usando la pseudo-clase :link.
  • Visited: Un enlace ha sido visitado (existe en el historial del navegador), se establece usando la pseudo-clase :visited.
  • Hover: Cuando se pasa el puntero del ratón sobre un enlace, se establece usando la pseudo-clase :hover.
  • Focus: Un enlace cuando ha sido focalizado (por ejemplo, al desplazarse un usuario con el teclado usando la tecla Tab o similar, o programáticamente centrado usando HTMLElement.focus()) - se establece usando la pseudoclase :focus.
  • Active: Un enlace cuando se está pulsando (por ejemplo, al hacer clic sobre él), se establece el diseño con la pseudoclase :active.

Los estilos por defecto pueden ser desactivados/modificados usando las siguientes propiedades CSS:

  • color para el color del texto.
  • cursor para el estilo del puntero del ratón - no debería desactivarlo a menos que tenga una buena razón.
  • outline para el contorno de texto (un contorno es similar a un borde, la única diferencia es que el borde ocupa espacio en el cuadro y un contorno no lo hace; sólo se coloca sobre la parte superior del fondo). La opción outline es una ayuda útil para la accesibilidad, así que piénsalo bien antes de desactivarlo; deberías al menos duplicar los estilos dados al estado del enlace hover y en el estado de focus también.

Orden de las pseudo-clases de los enlaces:


	a {
	}
	
	a:link {	
	}
	
	a:visited {
	}
	
	a:focus {
	}
	
	a:hover {
	}
	
	a:active {
	}

El orden es importante porque los estilos de enlace se construyen uno sobre el otro, por ejemplo, los estilos de la primera regla se aplicarán a todos los siguientes, y cuando un enlace se activa, también se pasa el puntero del ratón sobre él. Si los pones en el orden equivocado, las cosas no funcionarán correctamente. Para recordar el orden, puedes intentar usar una mnemotécnica como:
LoVe Fears HAte.

Para crear un menú a partir de una lista con enlaces nos debemos fijar en dos parametros principales, la opción display de los li y de los enlaces a:


	li {
	  display: inline; /* Para conseguir que se comporten como una linea. */
	}
	
	a {
	  display: inline-block; /* Esto permite manejar los enlaces como cajas */
	  line-height: 3;
	}

Fuentes para la WEB.

Las fuentes web son una función CSS que le permite especificar los archivos de fuentes que se descargarán junto con su sitio web a medida que se accede a él, lo que significa que cualquier navegador que admita fuentes web puede tener exactamente las fuentes que usted especifique a su disposición.:

Lo primero de todo, tenemos un bloque @font-face al comienzo del archivo CSS, que especifica que fichero(s) se descargaran:


	@font-face {
	  font-family: "myFont";
	  src: url("myFont.ttf");
	}

Debajo de esto puedes usar el nombre de la familia de fuentes especificada dentro de @font-face para aplicar tu fuente personalizada a lo que quieras, como siempre:


	html {
	  font-family: "myFont", "Bitstream Vera Serif", serif;
	}

Para añadir el código de fuentes que necesitamos, y los tipos de fuentes, podemos acceder a FontSquirrel. Con el fichero(s) de la fuente nos genera tanto los ficheros de tipos de fuentes compatibles y los archivos CSS que tendremos que añadir a nuestro sitio web. El código CSS será algo como:


	@font-face {
	  font-family: 'ciclefina';
	  src: url('fonts/cicle_fina-webfont.eot');
	  src: url('fonts/cicle_fina-webfont.eot?#iefix') format('embedded-opentype'),
	         url('fonts/cicle_fina-webfont.woff2') format('woff2'),
	         url('fonts/cicle_fina-webfont.woff') format('woff'),
	         url('fonts/cicle_fina-webfont.ttf') format('truetype'),
	         url('fonts/cicle_fina-webfont.svg#ciclefina') format('svg');
	  font-weight: normal;
	  font-style: normal;
	}

Tras incluir este código podemos hacer referencia a la fuente en el CSS con:


	html {
	  font-family: "ciclefina", serif;
	}

Uso de un servicio de fuentes en línea:

Los servicios de fuentes en línea generalmente almacenan y sirven fuentes para ti, así que no tienes que preocuparte por escribir el código @font-face, y generalmente sólo necesitas insertar una simple línea o dos de código en tu sitio para que todo funcione.

Lo primero es ir a GoogleFonts y allí seleccionar las fuentes que queremos utilizar. En la web nos darán un código que debemos añadir en la cabecera de nuestro código HTML que vaya a usarla.

<link href="https://fonts.googleapis.com/css?family=Sarabun" rel="stylesheet">

En la misma página de Google nos proporcionan el código que usaremos en nuestro CSS:

font-family: 'Sarabun', sans-serif;
