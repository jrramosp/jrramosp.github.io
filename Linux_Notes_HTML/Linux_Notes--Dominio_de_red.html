¿Cómo cambiar el dominio de red en Ubuntu?
Para cambiar el dominio de Ubuntu sólo tenemos que realizar los siguientes pasos:

Creamos un archivo de respaldo (por si acaso)
$sudo cp /etc/samba/smb.conf /etc/samba/smb.conf_respaldo

Ahora modificaremos el archivo para agregar el dominio deseado
$sudo gedit /etc/samba/smb.conf

Buscamos la línea que diga
...
workgroup=WORKGROUP
...

La reemplazamos con el nombre de dominio deseado

workgroup=DOMINIO

Guardamos el archivo y finalizamos con lo siguiente en la consola

$sudo testparm
$sudo /etc/init.d/samba restart
