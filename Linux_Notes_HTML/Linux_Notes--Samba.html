Samba

Para compartir carpetas con nuestro usuario de red
En nautilus selecionar la carpeta que queremos compartir y botón derecho Compartir carpeta
Opción “Permitir a otras personas...”
Instalará Samba
Después de eso en la terminal sudo smbpasswd -a usuario pide un password para el usuario
Y eso es todo
Usar en android APP CX-Explorer.
Para el samba
sudo apt install nfs-common
sudo apt install smbclient
sudo apt install cifs-utils
