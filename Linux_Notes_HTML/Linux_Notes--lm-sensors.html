lm-sensors


Created domingo 18 junio 2017

Te contesto lo referente a la temperatura de la placa y velocidad de los ventiladores (el resto no lo conozco):

- Instala el paquete lm-sensors

- Ejecuta en consola:    sudo sensors-detect

   Contesta a todas las preguntas que si, incluida la última en la que la contestación por defecto es que no.

- O bien reinicias, o bien instalas todos los módulos que te ha detectado sensors-detect con: sudo modprobre nombre_del_modulo

- Ejecuta en consola sensors. Te debería salir la información de temperaturas, voltajes y ventiladores.

- Como supongo que querrás ver la información con una herramienta gráfica, instala alguna que la muestre, como gkrellmc (por defecto no muestra está información, hay que ir a configuración y activar las opciones correspondientes).

- Para el entorno KDE, existen varios temas de superkaramba que muestran  información del sistema en el fondo del escritorio. No se si hay algo equivalente para GNOME.

comandos para ventiladores

sudo apt-get install fancontrol
pwmconfig

http://ubuntuforums.org/showthread.php?t=250025

A partir de la version 10.10 ubuntu

sudo gedit /etc/default/grub

And enter the following line:

GRUB_CMDLINE_LINUX="acpi_enforce_resources=lax"

ejecutamos pwmcontrol y preparamos la confiuración
aparecen 5 controladores de configuración, nos quedamos con el primero
configuracion que queda en sudo gedit /etc/fancontrol

# Configuration file generated by pwmconfig, changes will be lost
INTERVAL=5
DEVPATH=hwmon0=devices/platform/w83627ehf.656
DEVNAME=hwmon0=w83627ehf
FCTEMPS=hwmon0/device/pwm1=hwmon0/device/temp1_input hwmon0/device/pwm2=hwmon0/device/temp2_input
FCFANS=hwmon0/device/pwm1= hwmon0/device/pwm2=hwmon0/device/fan2_input
MINTEMP=hwmon0/device/pwm1=35 hwmon0/device/pwm2=55
MAXTEMP=hwmon0/device/pwm1=45 hwmon0/device/pwm2=68
MINSTART=hwmon0/device/pwm1=150 hwmon0/device/pwm2=150
MINSTOP=hwmon0/device/pwm1=100 hwmon0/device/pwm2=100

Finalmente lo añadimos a sudo gedit /etc/rc.local
/etc/init.d/fancontrol start
exit 0
