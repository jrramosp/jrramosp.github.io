Activar y desactivar el touchpad en Lubuntu

Run the following command in a terminal:

xinput list

It displays all the input devices connected. Note that they all have an id. Since 12 is the id for my touchpad, running the following command will disable it.

xinput set-prop 14 "Device Enabled" 0

or

synclient TouchpadOff=1

or a√±adir al autostart

syndaemon -d -i 3 -t

Syndaemon supports the following options (flags):

-i <idle-time> How many seconds to wait after the last key press before enabling the touchpad. (default is 2s).
-d Start as a daemon, ie in the background.
-t Only disable tapping, not mouse movements, in response to keyboard activity.
