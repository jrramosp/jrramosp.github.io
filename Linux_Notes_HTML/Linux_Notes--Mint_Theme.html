Mint Theme


Created domingo 18 junio 2017

Si por el contrario prefieres utilizar el nuevo repositorio PPA provisto por WebUpd8 para instalar y mantener actualizado el tema de forma automática, ejecuta las siguientes lineas en la terminal:

sudo add-apt-repository ppa:webupd8team/themes
sudo apt-get update
sudo apt-get install mint-x-icons minty-freshness-theme
