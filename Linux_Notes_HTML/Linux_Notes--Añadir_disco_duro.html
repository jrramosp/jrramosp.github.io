Añadir disco duro



Formatear y particionar el disco en ext4 .
Crear una carpeta de montaje /media/usuario/disco
Añadir en fstab el UUID
UUID=4a243874-0319-410a-9abc-dad7b891c1af /media/nani/Teradisc ext4 auto,user,rw,x-gvfs-name=Teradisc,x-gvfs-show 0 0
Cambiar permisos en la carpeta de montaje
sudo chown -R nani /media/nani/Teradisc/
sudo chmod -R ugo+rw /media/nani/Teradisc

Asignar nueva carpeta en el sistema
