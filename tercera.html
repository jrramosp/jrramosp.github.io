Las tablas en HTML deben ser usadas de forma muy específica para mostrar datos ordenados. Así para introducir una tabla en nuestra página usaremos la siguiente estructura; table abre la tabla.tr indica la primera fila y td donde se añaden los datos de las celdas. Si los datos son cabecera de fila o columna podemos indicarlo con th.


        <table>
                <tr>
                     <th></th>
                     <td></td>
                </tr>
        </table>

Si queremos que nuestras celdas ocupen varias celdas usaremos colspan o rowspan.

La etiqueta colgroup y col especifican un grupo o más de columnas en una tabla para darle estilos.


        <table>
          <colgroup>
            <col>
            <col style="background-color: yellow" span="2">
          </colgroup>
          <tr>...

Si queremos especificar más de una columna usaremos el atributo span

También podemos añadir un título a la tabla con la etiquetacaption.


        <table>
          <caption>Dinosaurs in the Jurassic period</caption>
        
          ...
        </table>

Otra parte importante de la tabla es estructurar los datos con las etiquetas thead, tbody y tfoot. Las etiquetas son por si mismas muy explicativas de las partes a las que corresponden en la tablas.


        <table>
                <thead>
                        <th>Cabecera</th>
                </thead>
                <tbody>
                        <td>Valor 1</td>
                        <td>Valor 2</td>
                </tbody>
                <tfoot>
                        <td>Pié de tabla</td>
                </tfoot>
        </table>

Con el fin de ayudar a los usuarios que utilizan lectores de pantallas se deben añadir una serie de etiquetas que hagan indicaciones de los datos a los que se está accediendo. Para ello añadiremos el atributo scope a las etiquetas th indicando así la funcion que cumple en la tabla.

scope = "row" para indicar que es cabecera de una fila o scope = "col" si es para indicar una cabecera de columna. Además tiene otros dos valores si indican cabeceras sobre grupos de filas o columnas. Estos valores son scope = "colgroup" y scope = "rowgroup".

Hay una forma alternativa para identificar las celdas con el atributo id y el atributo headers. Primero identificaremos cada th con un único id. Depués pondremos en cada td el atributo headers con los valores de cada th que coincidan con esa celda.


        <thead>
          <tr>
            <th id="purchase">Purchase</th>
            <th id="location">Location</th>
            <th id="date">Date</th>
            <th id="evaluation">Evaluation</th>
            <th id="cost">Cost (€)</th>
          </tr>
        </thead>
        <tbody>
        <tr>
          <th id="haircut">Haircut</th>
          <td headers="location haircut">Hairdresser</td>
          <td headers="date haircut">12/09</td>
          <td headers="evaluation haircut">Great idea</td>
          <td headers="cost haircut">30</td>
        </tr>
        
          ...
        
        </tbody>
