Añadiendo autor y descripción


	<meta name="author" content="Juan Ramón">
	<meta name="description" content="Aquí se pone la descripción de la página web para que aparezca en los buscadores">

Añadiendo un icono a la página(favicon), un enlace a un estilo CSS y un enlace a un script


	<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
	<link rel="stylesheet" href="css/style.css">
	<script type="text/javascript" src="script.js"></script>

Para estructurar los documentos debeos utilizar los elemento HEADING: h1,h2,h3,h4,h5,h6

Usar listas, tanto numerales como listas abstractas 


	<ul>
		<li></li>
	</ul>
	<ol>
		<li></li>
	</ol>

Tambien definimos énfasis con <em></em> e importacia con <strong></strong>

  • <i> Se usa principalmente como itálica para: Palabras extranjeras, definiciones taxonomicas, terminos técnicos...
  • <b> Se usa principalmente como negrita para: palabras clave, nombre de productos...
  • <u> Se usa principalmente como subrayado para: nombre propio, error de escritura...

También podemos definir listas con definiciones, con la descripción primero y luego la definición


	<dl>
		<dt>Definición</dt>
		<dd>Descripción</dd>
	</dl>

Los enlaces deben contener descripciones lógicas (no un "Haz click aquí"), y son:

	<a href="mailto:nobody@mozilla.org?subject=Correo"></a>

Si citamos en nuestro texto en un elemento bloque lo haremos con


	<blockquote cite="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/blockquote">
  	<p>The <strong>HTML <code>&lt;blockquote&gt;</code> Element</strong> (or <em>HTML Block Quotation Element</em>) indicates that the enclosed
  	text is an extended quotation.</p>
	</blockquote>

Si citamos en un elemento lineal lo haremos


	<p>The quote element — <code>&lt;q&gt;</code> — is <q cite="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/q">
	intended for short quotations that don't require paragraph breaks.</q></p>

Además tenemos el elemento cite para remarcar la cita


	<p>According to the <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/blockquote">
	<cite>MDN blockquote page</cite></a>:
	</p>
