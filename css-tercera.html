Fondos de elementos: El fondo de un elemento es el área que se encuentra debajo del contenido, relleno y borde de un elemento.Hay cuatro propiedades básicas:

background-color que determina un color para el fondo. El color por defecto es transparent.

background-image para determinar una imagen de fondo con la función url(). Por defecto las imágenes se repiten horizontal y verticalmente.Hay que aclarar que solo se utilizan como imágenes decorativas.

background-repeat determina como se repite el fondo en nuestro elemento. Puede tomar los valores:

  • no-repeat: La imagen no se repetirá en absoluto: sólo se mostrará una vez.
  • repeat-x: La imagen se repetirá horizontalmente hasta el fondo.
  • repeat-y: La imagen se repetirá verticalmente hasta el fondo.
  • repeat: La imagen se repetirá tanto vertical como horizontalmente.

background-position permite posicionar su imagen de fondo donde quiera dentro del fondo. Generalmente la propiedad tomará dos valores separados por un espacio, que especifican las coordenadas horizontales (x) y verticales (y) de la imagen. La propiedad puede aceptar muchos tipos de valor diferentes que se pueden combinar. Los más comunes que usará son:

  • Absolutos (como pixels)
  • Relativos (como rem, em)
  • Porcentuales
  • Palabras clave: Estos dos valores son intuitivos, y pueden tomar valores de left, center, right, y top, center, bottom, respectivamente.

Gradientes en los fondos.Hay dos tipos de gradientes disponibles en este momento: gradientes lineales (que van de una línea recta a otra) y gradientes radiales (que irradian desde un solo punto).

Un gradiente lineal se crea pasando en una función linear-gradient() como el valor de una propiedad de background-image. Como mínimo, la función necesita tomar tres parámetros separados por comas - la dirección en la que el degradado debe ir a través del fondo, el color al principio y el color al final.

background-image: linear-gradient(to bottom, orange, yellow);

Puede utilizar palabras clave para especificar la dirección (to bottom, to right, to bottom right, etc.), o los valores de grado (0deg es equivalente a to top, 90deg es equivalente a to right, hasta 360deg, lo que equivaldría a to top de nuevo). También puede especificar otros puntos a lo largo del degradado donde se define el color - estos se llaman topes de color, y el navegador calculará los topes de color entre cada conjunto de topes de color.

background-image: linear-gradient(to bottom, yellow, orange 40%, yellow);

Tenga en cuenta que también puede establecer un gradiente lineal de repetición utilizando la función repeating-linear-gradient(). Esto funciona exactamente de la misma manera, excepto que el patrón que usted establece se repite una y otra vez hasta el borde del fondo.

background-image: repeating-linear-gradient(to right, yellow, orange 25px, yellow 50px);

Otra opción que tenemos disponible para los fondos es especificar cómo se desplazan cuando el contenido se desplaza. Esto se controla usando la propiedad background-attachment, que puede tomar los siguientes valores:

  • scroll: Hace que el fondo del elemento se desplace cuando se desplaza la página. Si el contenido del elemento se desplaza, el fondo no se mueve. En efecto, el fondo se fija en la misma posición de la página, por lo que se desplaza como se desplaza la página.
  • fixed: Hace que el fondo de un elemento se fije en la ventana de visualización, de modo que no se desplace cuando se desplaza el contenido de la página o del elemento. Siempre permanecerá en la misma posición en la pantalla.
  • local: Este valor se añadió más tarde (sólo se admite en Internet Explorer 9+, mientras que los demás se admiten en IE4+) porque el valor de desplazamiento es bastante confuso y no hace realmente lo que se desea en muchos casos. El valor local fija el fondo al elemento en el que está fijado, de modo que cuando se desplaza el elemento, el fondo se desplaza con él.

Es posible declarar todos los valores de propiedad discutidos anteriormente (y algunos otros, en navegadores más nuevos) en una sola línea, usando la propiedad background. Para ello, se utilizan los mismos valores que en las propiedades individuales, pero se colocan todos en una línea separada por espacios, así:

background: yellow linear-gradient(to bottom, orange, yellow) no-repeat left center scroll;

Múltiples fondos: Recientemente (desde Internet Explorer 9) hemos tenido la capacidad de adjuntar múltiples fondos a un solo elemento. Esto es bueno, ya que múltiples fondos son muy útiles. Se separan las diferentes definiciones de fondo con comas:


	background: url(image.png) no-repeat 99% center,
	            url(background-tile.png),
	            linear-gradient(to bottom, yellow, #dddd00 50%, orange);
	background-color: yellow;

También hay que tener en cuenta que hemos puesto el color de fondo de reserva en una declaración de propiedad separada, porque tratar de incluirlo en los múltiples fondos parece romper las cosas.

También se puede poner múltiples valores en las propiedades de background-*, por ejemplo:


	background-image: url(image.png), url(background-tile.png);
	background-repeat: no-repeat, repeat;

Como mencionamos anteriormente, hay una propiedad disponible - background-size - que le permite alterar dinámicamente el tamaño de una imagen de fondo para que se ajuste mejor a su diseño. Esto es muy útil en muchos sentidos, por ejemplo, para corregir automáticamente el tamaño de los iconos que no se cargan correctamente. Para cada imagen de fondo, necesita incluir dos valores de tamaño de fondo - uno para la dimensión horizontal y otro para la vertical:

background-size: 16px 16px;

border: La propiedad abreviada border le permite establecer los cuatro lados a la vez. Pero border se puede dividir en numerosas etiquetas diferentes para necesidades de estilo más específicas:

  • border-top, border-right, border-bottom, border-left: Establece grosor, estilo y color para cada lado de un elemento.
  • border-width, border-style, border-color: Ajustar sólo el grosor, estilo o color individualmente, pero para los cuatro lados del borde.
  • También puede establecer una de las tres propiedades de un solo lado del borde individualmente, usando border-top-width, border-top-style, border-top-color, etc.

Las propiedades largas también están disponibles, para establecer el radio del borde de cada esquina de la caja individualmente: border-top-left-radius, border-top-right-radius, border-bottom-left-radius, y border-bottom-right-radius.

Consejos rápidos para el diseño de tablas

  • Haga que el marcado de su tabla sea lo más sencillo posible y mantenga la flexibilidad, por ejemplo, utilizando porcentajes, para que el diseño sea más receptivo.
  • Usar table-layout: fixed para crear un diseño de tabla más predecible que le permita establecer fácilmente los anchos de columna estableciendo el ancho en sus encabezados (<th>).
  • Usar border-collapse: collapse para hacer que los bordes de los elementos de la tabla colapsen entre sí, produciendo una apariencia más ordenada y fácil de controlar.
  • Utilice <thead>, <tbody>, y <tfoot> para dividir su tabla en trozos lógicos y proporcionar lugares adicionales para aplicar CSS, de modo que sea más fácil superponer estilos si es necesario.
  • Use rayas de cebra para hacer que las filas alternativas sean más fáciles de leer.
  • Use text-align para alinear su texto <th> y <td>, para hacer las cosas más ordenadas y fáciles de seguir.

    box-shadow: 5px 5px 5px rgba(0,0,0,0.7);

    También puede especificar varias box-shadow en una sola declaración, separándolas con comas. box-shadow tiene disponible una palabra clave inset. Ponerla al principio de una declaración de sombra hace que se convierta en una sombra interior

    box-shadow: inset 2px 2px 1px black;

    Los filtros CSS proporcionan una forma de aplicar filtros a un elemento de la misma manera que se puede aplicar un filtro a una capa en un paquete gráfico como Photoshop.

    	.filter {
	  -webkit-filter: drop-shadow(5px 5px 1px rgba(0,0,0,0.7));
	  filter: drop-shadow(5px 5px 1px rgba(0,0,0,0.7));
	}

    Los modos de mezcla CSS nos permiten añadir modos de mezcla a los elementos que especifican un efecto de mezcla cuando dos elementos se superponen - el color final mostrado para cada píxel será el resultado de una combinación del color original del píxel, y el del píxel en la capa inferior.

    Hay dos propiedades que utilizan modos de mezcla en CSS: background-blend-mode, que combina múltiples imágenes de fondo y colores en un solo elemento, y mix-blend-mode, que mezcla el elemento en el que está colocado con los elementos que se superponen - tanto el fondo como el contenido. Los valores que pueden tomar son muy amplios.
