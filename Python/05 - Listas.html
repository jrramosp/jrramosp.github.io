05 - Listas

Introducción a las listas:

Las listas son un tipo de datos que se pueden utilizar para almacenar una colección de diferentes piezas de información como una secuencia bajo un solo nombre de variable. (Los tipos de datos sobre los que ya has aprendido incluyen cadenas, números y booleanos).
Puede asignar elementos a una lista con una expresión de la forma:

list_name = [item_1, item_2]

con los artículos entre paréntesis. Una lista también puede estar vacía:

lista_vacía = []

Las listas son muy similares a las cadenas de texto, pero hay algunas diferencias clave.
¡Los índices de la lista comienzan con 0, no con 1! Accede al primer elemento de una lista como esta: nombre_de_lista[0]. El segundo elemento de una lista está en el índice 1: nombre_lista[1]
Una lista no tiene que tener una longitud fija. ¡Puedes añadir elementos al final de una lista cuando quieras!

letters = ['a', 'b', 'c']
letters.append('d')
print (len(letters))
print (letters)
letters = ['a', 'b', 'c']
letters.append('d')
print (len(letters))
print (letters)
  • En el ejemplo anterior, primero creamos una lista llamada letters.
  • Luego, añadimos la cadena "d" al final de la lista letters.
  • A continuación, imprimimos 4, la longitud de la lista letters.
  • Finalmente, imprimimos ['a', 'b', 'c', 'd'].
    El operador + concatena listas

Dividir la lista

A veces, sólo quieres acceder a una parte de la lista.

letters = ['a', 'b', 'c', 'd', 'e']
slice = letters[1:3]
print (slice)
print (letters)
letters = ['a', 'b', 'c', 'd', 'e']
slice = letters[1:3]
print (slice)
print (letters)
  • En el ejemplo anterior, primero creamos una lista llamada letters.
  • Luego, tomamos una subsección y la almacenamos en slice. Comenzamos en el índice antes del colon y continuamos hasta, pero sin incluir, el índice después del colon.
  • A continuación, imprimimos ['b', 'c']. Recuerden que empezamos a contar los índices desde 0 y que nos detuvimos antes del índice 3.
  • Finalmente, imprimimos ['a', 'b', 'c', 'd', 'e'], sólo para mostrar que no modificamos la lista letters original.

¡Puedes cortar un texto exactamente como una lista! De hecho, puedes pensar en las cadenas de texto como listas de caracteres: cada carácter es un elemento secuencial de la lista, a partir del índice 0.

my_list[:2]
# Obtiene los dos primero elementos
my_list[3:]
# Obtiene desde el cuarto al final de los elementos
my_list[:2]
# Obtiene los dos primero elementos
my_list[3:]
# Obtiene desde el cuarto al final de los elementos

Si la división de la lista incluye el primer o el último elemento de una lista (o una cadena), el índice de ese elemento no tiene que ser incluido.
La función de rango tiene tres versiones diferentes:

range(stop)
range(start, stop)
range(start, stop, step)

En todos los casos, la función range() devuelve una lista de números desde el inicio hasta (pero sin incluir) la parada. Cada elemento se incrementa por pasos.
Si se omite, el valor por defecto del inicio es 0 y el del paso es 1.

Mantener el orden

A veces hay que buscar un elemento en una lista.

animals = ["ant", "bat", "cat"]
print (animals.index("bat"))
animals = ["ant", "bat", "cat"]
print (animals.index("bat"))
  • Primero, creamos una lista llamada animals con tres textos.
  • Luego, imprimimos el primer índice que contiene la cadena "bat", que imprimirá 1.

También podemos insertar elementos en una lista.

animals.insert(1, "dog")
print (animals)
animals.insert(1, "dog")
print (animals)
  • Insertamos "dog" en el índice 1, que hace que todo se mueva hacia abajo en 1.
  • Imprimimos ["ant", "dog", "bat", "cat"]
    extend toma una lista como argumento y agrega todos los elementos:
>>> t1 = ['a', 'b', 'c']
>>> t2 = ['d', 'e']
>>> t1.extend(t2)
>>> print(t1)
['a', 'b', 'c', 'd', 'e']

>>> t1.extend(t2)

Si tu lista es un lío, puede que tengas que ordenarla.

animals.sort()

A veces hay que quitar algo de una lista.

beatles = ["john","paul","george","ringo","stuart"]
beatles.remove("stuart")
print (beatles)
beatles = ["john","paul","george","ringo","stuart"]
beatles.remove("stuart")
print (beatles)

Este código imprime:

["john","paul","george","ringo"]
  • Creamos una lista llamada beatles con 5 cadenas de texto.
  • Luego, quitamos el primer elemento de los beatles que coincida con la cadena "stuart". Note que .remove(item) no devuelve nada.
  • Finalmente, imprimimos esa lista sólo para ver que "stuart" fue realmente eliminado.

Eliminar elementos de las listas

  • n.pop(index) eliminará el artículo en el índice de la lista y se lo devolverá:
    n = [1, 3, 5]
    n.pop(1)
    # Returns 3 (the item at index 1)
    print (n)
    # prints [1, 5]
    n = [1, 3, 5]
    n.pop(1)
    # Returns 3 (the item at index 1)
    print (n)
    # prints [1, 5]
  • n.remove(item) eliminará el artículo actual si lo encuentra:
    n.remove(1)
    # Removes 1 from the list,
    # NOT the item at index 1
    print (n)
    # prints [3, 5]
    n.remove(1)
    # Removes 1 from the list,
    # NOT the item at index 1
    print (n)
    # prints [3, 5]
  • del(n[1]) es como .pop in que eliminará el elemento en el índice dado, pero no lo devolverá:
    del(n[1])
    # Doesn't return anything
    print (n)
    # prints [1, 5]
    del(n[1])
    # Doesn't return anything
    print (n)
    # prints [1, 5]

Para remover más de un elemento, puedes usar del con un índice de rebanado:
del t[1:5]

Iterar sobre una lista en una función

Método 1 - para el artículo de la lista:

for item in list:
  print (item)
for item in list:
  print (item)

Método 2 - iterar a través de los índices:

for i in range(len(list)):
  print (list[i])
for i in range(len(list)):
  print (list[i])

El método 1 es útil para hacer un bucle en la lista, pero no es posible modificar la lista de esta manera.
El método 2 utiliza índices para hacer un bucle en la lista, lo que permite también modificar la lista si es necesario.

Construcción de listas

Digamos que quisieras construir una lista de los números del 0 al 50 (inclusive). Podríamos hacer esto muy fácilmente:

mi_lista = rango(51)

Pero, ¿y si quisiéramos generar una lista según alguna lógica, por ejemplo, una lista de todos los números pares del 0 al 50?
La respuesta de Python a esto es la comprensión de la lista. Las comprensiones de listas son una poderosa forma de generar listas usando las palabras clave "for/in" y "if" que hemos aprendido.

evens_to_50 = [i for i in range(51) if i % 2 == 0]
print (evens_to_50)
evens_to_50 = [i for i in range(51) if i % 2 == 0]
print (evens_to_50)

Si sólo quisieras los números duplicados que son divisibles por igual entre tres:

doubles_by_3 = [x * 2 for x in range(1, 6) if (x * 2) % 3 == 0]
# => [6]
doubles_by_3 = [x * 2 for x in range(1, 6) if (x * 2) % 3 == 0]
# => [6]

Sintaxis de corte de lista

A veces sólo queremos una parte de la lista de Python. Tal vez sólo queremos los primeros elementos; tal vez sólo queremos los últimos. ¡Quizás queremos todos los demás elementos!
El rebanado de listas nos permite acceder a los elementos de una lista de manera concisa. La sintaxis es así:
[inicio:fin:paso]
Donde empieza describe donde empieza la rebanada (inclusivo), donde termina (exclusivo), y zancada describe el espacio entre los elementos de la lista de rebanadas. Por ejemplo, una zancada de 2 seleccionaría cada uno de los otros elementos de la lista original para colocarlos en la lista de rebanadas.

Funciones para listas

Hay un cierto número funciones internas que pueden ser utilizadas en las listas que te permiten mirar rápidamente a través de una lista sin escribir tus propios bucles. La función sum() solamente funciona cuando los elementos de la lista son números. Las funciones (max(), min(),len(), etc.) funcionan con listas de cadenas y otros tipos que pueden ser comparados entre sí.
Listas y cadenas
Una cadena es una secuencia de caracteres y una lista es una secuencia de valores, pero una lista de caracteres no es lo mismo que una cadena. Para convertir una cadena en una lista de caracteres, puedes usar list.
La función list divide una cadena en letras individuales. Si quieres dividir una cadena en palabras, puedes utilizar el método split.
Puedes llamar split con un argumento opcional llamado delimitador que especifica qué caracteres usar para delimitar las palabras. El siguiente ejemplo utiliza un guión medio como delimitador:

>>> s = 'spam-spam-spam'
>>> delimiter = '-'
>>> s.split(delimiter)
['spam', 'spam', 'spam']
>>> s = 'spam-spam-spam'
>>> delimiter = '-'
>>> s.split(delimiter)
['spam', 'spam', 'spam']

join es el inverso de split. Este toma una lista de cadenas y concatena los elementos. join es un método de cadenas, así que tienes que invocarlo en el delimitador y pasar la lista como un parámetro:

>>> t = ['suspirando', 'por', 'los', 'fiordos']
>>> delimiter = ' '
>>> delimiter.join(t)
'suspirando por los fiordos'
>>> t = ['suspirando', 'por', 'los', 'fiordos']
>>> delimiter = ' '
>>> delimiter.join(t)
'suspirando por los fiordos'

Listas como argumentos

Cuando pasas una lista a una función, la función obtiene un apuntador a la lista. Si la función modifica un parámetro de la lista, el código que ha llamado la función también verá el cambio. Por ejemplo, remover_primero elimina el primer elemento de una lista:

def remover_primero(t):
    del t[0]Aquí está el ejemplo de cómo se usa:
>>> letras = ['a', 'b', 'c']
>>> remover_primero(letras)
>>> print(letras)
['b', 'c']
def remover_primero(t):
    del t[0]Aquí está el ejemplo de cómo se usa:
>>> letras = ['a', 'b', 'c']
>>> remover_primero(letras)
>>> print(letras)
['b', 'c']

El parámetro t y la variable letras son alias para el mismo objeto.
Es importante distinguir entre operaciones que modifican listas y operaciones que crean nuevas listas. Por ejemplo, el método append modifica una lista, pero el operador + crea una nueva lista:

>>> t1 = [1, 2]
>>> t2 = t1.append(3)
>>> print(t1)
[1, 2, 3]
>>> print(t2)
None

>>> t3 = t1 + [3]
>>> print(t3)
[1, 2, 3]
>>> t2 is t3
False
>>> t1 = [1, 2]
>>> t2 = t1.append(3)
>>> print(t1)
[1, 2, 3]
>>> print(t2)
None

>>> t3 = t1 + [3]
>>> print(t3)
[1, 2, 3]
>>> t2 is t3
False

Esta diferencia es importante cuando escribes funciones que no están destinadas a modificar listas. Por ejemplo, esta función no elimina el primer elemento de una lista:

def mal_eliminar_primero(t):
    t = t[1:]              # ¡EQUIVOCADO!
def mal_eliminar_primero(t):
    t = t[1:]              # ¡EQUIVOCADO!

El operador de rebanado crea una nueva lista y el asignamiento hace que t apunte a la lista, pero nada de esto tiene efecto en la lista que fue pasada como argumento.
Una alternativa es escribir una función que cree y regrese una nueva lista. Por ejemplo, cola regresa todo excepto el primer elemento de una lista:

def cola(t):
    return t[1:]
def cola(t):
    return t[1:]
