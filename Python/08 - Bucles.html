08 - Bucles

Bucle

El bucle while es similar al declaración if: ejecuta el código en su interior si la condición es True. La diferencia es que el bucle while continuara la ejecución mientras que la condición sea cierta. En otras palabras, en lugar de ejecutar si algo es true, se ejecuta mientras la condición es true.

loop_condition = True

while loop_condition:
  print ("I am a loop")
  loop_condition = False
loop_condition = True

while loop_condition:
  print ("I am a loop")
  loop_condition = False

Break

El break es una declaración de “una linea” que significa “salir del bucle actual”. Una forma alternativa para salir de nuestro bucle contador y parar su ejecución es con la declaración break.

  1. Primero, crea un while con una condición que siempre sea true. Se muestra la forma más simple.
  2. Usando una declaración if, defines la condición de parada. Dentro del if, escribes el break, con el significado “salir del loop”.
    La diferencia aquí es que el bucle se garantiza el correr al menos una vez.
    Continue
    Algunas veces, estando dentro de un bucle se necesita terminar con la iteración actual y saltar a la siguiente de forma inmediata. En ese caso se puede utilizar la sentencia continue para pasar a la siguiente iteración sin terminar la ejecución del cuerpo del bucle para la actual.
while True:
    linea = input('> ')
    if linea[0] == '#' :
        continue
    if linea == 'fin':
        break
    print(linea)
print('¡Terminado!')

        continue
    if linea == 'fin':
        break
    print(linea)
print('¡Terminado!')

While / else

Algo completamente diferente sobre Python es la construcción while/else. while/else es similiar a if/else, pero hay una diferencia: el bloque else se ejecutará en el momento en que la condición del bucle sea evaluada a false.. Esto significa que se ejecutará si el bucle nunca entra o si el bucle sale con normalidad. Si del bucle se sale como resultado de un break, la condición else no se ejecutará.

import random

print ("Lucky Numbers! 3 numbers will be generated.")
print ("If one of them is a '5', you lose!")

count = 0
while count < 3:
  num = random.randint(1, 6)
  print (num)
  if num 5:
    print ("Sorry, you lose!")
    break
  count += 1
else:
  print ("You win!")

For

Una alternativa de bucle en el bucle for. La sintaxis es como se muestra en el código. El ejempo significa “por cada numero en el rando 0-9, imprime i”.

for i in range(10):
  print (i)
for i in range(10):
  print (i)

Este tipo de bucle es útil cuando quieres hacer algo un número determinado de veces, como añadir algo al final de una lista.
Quizás la forma más útil ( y la más común) de uso de el bucle for es recorrer una lista.

contador = 0

Iterando sobre un diccionario

Te preguntaras como funciona iterar sobre un diccionario. Obtendrás la clave o el valor?
La respuesta corta es: Tienes la clave que usaras para obtener el valor.

d = {'x': 9, 'y': 10, 'z': 20}
for key in d:
  if d[key] == 10:
    print ("This dictionary has the value 10!")
d = {'x': 9, 'y': 10, 'z': 20}
for key in d:
  if d[key] == 10:
    print ("This dictionary has the value 10!")
  1. Primero, creamos un diccionario con cadenas como claves y número como los valores.
  2. Entonces recorremos a través del diccionario, cada vez almacenando la llave en key.
  3. Después, comprobamos el valor de key es igual a 10.
  4. Si es así, imprimimos "This dictionary has the value 10!"

Contando según avanzas

Una debilidad de usar for-each bucles de iteración es que tú no sabes el índice del elemento al que miras. Generalmente esto no es el asunto, pero a veces es útil para saber cuan lejos en la lista estás. Por suerte la función enumerate ayuda con esto.
enumerate trabaja suministrando el índice correspondiente de cada elemento en la lista por la que pasas. Cada vez que vas a través del bucle, el índice sera uno más grande, y el elemento será el próximo elemento en la secuencia. Es muy similar a usar un bucle for con una lista, con la excepción de que nos da una manera más fácil de contar cuantos elementos hemos visto hasta ahora.

choices = ['pizza', 'pasta', 'salad', 'nachos']

print ('Your choices are:')
for index, item in enumerate(choices):
  print ((index + 1), item)
choices = ['pizza', 'pasta', 'salad', 'nachos']

print ('Your choices are:')
for index, item in enumerate(choices):
  print ((index + 1), item)

Múltiples listas

Es también común necesitar iterar sobre dos listas de inmediato. Aquí es donde la función zip nos viene a ayudar.
zip creará pares de elementos cuando se le pasan dos listas, y parará al final de la más corta.
zip puede manejar tres o mas listas también!!

list_a = [3, 9, 17, 15, 19]
list_b = [2, 4, 8, 10, 30, 40, 50, 60, 70, 80, 90]

For / else

Parecido a while, los bucles for pueden tener un else asociado con él.
En este caso, la declaración else se ejecuta después del for, pero solo si el for acaba normalmente, esto es, no con un break.

fruits = ['banana', 'apple', 'orange', 'tomato', 'pear', 'grape']

print ('You have...')
for f in fruits:
  if f == 'tomato':
    print ('A tomato is not a fruit!') # (It actually is.)
    continue
  print ('A', f)
else:
  print ('A fine selection of fruits!')
